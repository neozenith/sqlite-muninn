# Agentic Skills for sqlite-muninn

Use the [Vercel Labs `skills`](https://skills.sh/) cli to use this libraries skills:

```sh
npx skills add neozenith/sqlite-muninn
```

This will provide an interactice prompt to install these skills. Alternatively the following are one liners:


```sh
# Global
npx skills add neozenith/sqlite-muninn --agent claude-code codex --global --yes

# Project only
npx skills add neozenith/sqlite-muninn --agent claude-code codex --yes
```
